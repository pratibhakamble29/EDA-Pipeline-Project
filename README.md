# 📊 Data Cleaning & EDA Pipeline

## 🔹 Project Overview

This project demonstrates a complete data cleaning and exploratory data analysis (EDA) pipeline using Python.

## 🔹 Features

* Handling missing values
* Removing duplicates
* Fixing data types
* Outlier treatment using IQR
* Feature engineering
* Data visualization using Seaborn & Matplotlib

## 🔹 Tools Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib

## 🔹 Output

* Cleaned dataset (CSV)
* Multiple visualizations (PNG)
